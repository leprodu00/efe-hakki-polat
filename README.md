# Efe Hakkı Polat — Portfolio

Portfolio éditorial et immersif d’Efe Hakkı Polat, attaquant franco-turc.

## Direction créative

- Expérience cinématique rouge, noir et ivoire
- Scroll inertiel avec Lenis
- Animations au défilement avec Motion
- Galerie photo, parcours et présentation du joueur
- Design responsive et accessible

## Développement

```bash
npm install
npm run dev
```

Ouvrir ensuite [http://localhost:3000](http://localhost:3000).

## Vérification et production

```bash
npm run lint
npm run build
```

Le projet est développé avec Next.js et prêt à être déployé sur Vercel.
